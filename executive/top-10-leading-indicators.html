Leading Indicators Documentation

Overview

This dashboard displays the top 20 leading indicators that directly impact contact center operations. These metrics are predictive - they indicate potential issues before they affect customer experience. The indicators are grouped into four categories: Platform Health (1-10), Agent Staffing (11-14), Agent Tools (15-17), and Citrix Virtual Apps (18-20).

Data Sources

System Data Provided Refresh Rate API/Method
Genesys Cloud Platform health, queue metrics, voice quality, agent states Real-time (5s) Analytics API, Notifications API
Nuance ASR accuracy, IVR containment, recognition logs 5 minutes Nuance Analytics Export
ThousandEyes Network latency, packet loss, path visualization 1 minute ThousandEyes API v7
Nexthink DEX scores, endpoint health, application performance 15 minutes Nexthink NXQL API
SolarWinds Server CPU, memory, disk, network interface stats 1 minute SolarWinds SWIS API
In-House Apps CTI events, SSO logs, middleware health, KB availability Real-time Internal REST APIs
WFM System Schedule adherence, forecasts, shrinkage, staffing gaps Real-time (15s) WFM RTA API, Intraday Reports
Citrix Director Session availability, ICA latency, VDA health, failures 1 minute Citrix OData API, Director Monitoring

Calculation Methods

Indicator Formula Threshold (Target)
1. Genesys Availability ((Total Min - Downtime) / Total Min) × 100 ≥ 99.9%
2. Network Latency Avg RTT from agent sites to Genesys Cloud < 120ms
3. ASR Accuracy (Correct Recognitions / Total Attempts) × 100 ≥ 92%
4. DEX Score Weighted: Boot(15%) + App(25%) + Crash(20%) + Mem(15%) + CPU(15%) + Net(10%) ≥ 8.0/10
5. Queue Wait Time Σ(Wait Time for Answered) / Answered Count < 2:00
6. Infrastructure Health 100 - MAX(CPU%, Memory%, Disk%) ≥ 20% headroom
7. CTI Success Rate (Successful Screen Pops / Inbound Calls) × 100 ≥ 99%
8. MOS Score R-Factor based: MOS = 1 + 0.035R + R(R-60)(100-R)×7×10⁻⁶ ≥ 4.0
9. Login Success (Successful Logins / Total Attempts) × 100 ≥ 99.5%
10. Data Freshness Current Time - Last Update Timestamp < 15 seconds
Agent Staffing & Workforce
11. Schedule Adherence (Time in Scheduled Activity / Total Scheduled Time) × 100 ≥ 90%
12. Agent Occupancy (Handle Time / (Handle Time + Available Time)) × 100 75-85%
13. Staffing Gap Actual Logged-In Agents - Forecasted Required Agents ±5 agents
14. Unplanned Shrinkage (Unplanned Absent Hours / Total Scheduled Hours) × 100 < 6%
Agent Tools & Applications
15. KB Availability (Successful Searches / Total Search Requests) × 100 ≥ 99.5%
16. CRM Response Time Σ(Page Load Times) / Total Page Loads < 3 seconds
17. Softphone Health 100 - (Crash Events / Active Sessions × 100) ≥ 99%
Citrix Virtual Apps
18. Session Availability (Successful Launches / Total Launch Attempts) × 100 ≥ 98%
19. ICA Latency (RTT) Round-Trip Time between endpoint and VDA < 60ms
20. VDA Resource Health 100 - MAX(Avg CPU%, Avg Memory%) across VDAs ≥ 85%
